# 电子课程表

## 食用说明

1. git clone 本仓库
2. 配置  [scheduleConfig.js](./js/scheduleConfig.js)
3. 根目录cmd运行 ```npm install```
4. 根目录cmd运行 ```npm run build```
5. 在out文件夹中找到构建好的程序即可使用
